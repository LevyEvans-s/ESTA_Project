#### 1.登录


**POST /user/login**

> request

```
username,password
```
> response

fail
```
{
    "status": 1,
    "msg": "密码错误"
}
```

success
```
{
    "status": 0,
    "data": {
        "id": 12,
        "username": "aaa",
        "token":"Bearer xxxx"
    }
}
```
